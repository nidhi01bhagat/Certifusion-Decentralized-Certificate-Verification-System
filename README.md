# Certifusion - Decentralized Certificate Generation & Verification System
 A blockchain-based system for secure, tamper-proof certificate generation and verification using Ethereum and IPFS.


## Overview
Certifusion is a decentralized certificate generation and verification system designed to ensure authenticity, security, and trust in digital documents. The system leverages blockchain technology and IPFS to prevent fraud, tampering, and reliance on centralized authorities.

This project was developed as a major academic project, focusing on building a real-world system with strong emphasis on architecture, research, and implementation.

## Key Features

* Tamper-proof certificate generation
* Blockchain-based verification system
* Decentralized storage using IPFS
* Secure and transparent validation process
* Web-based interface for issuing and verifying certificates


## System Architecture

```text
User → Frontend → Backend → Smart Contract (Ethereum)
                             ↓
                          IPFS Storage
```

* Frontend handles user interaction
* Backend processes certificate requests
* Smart contracts manage verification logic
* IPFS stores certificate data securely


## Tech Stack
* Frontend: HTML, CSS, JavaScript, React.js
* Backend: Node.js, Express.js
* Database: MongoDB
* Blockchain: Ethereum, Solidity
* Tools: Truffle, Hardhat, Ganache
* Storage: IPFS, Pinata
* Wallet: MetaMask
